Aspiration

"열정적인 웹 퍼블리셔의 핵심인재"
어려서부터 계획을 설정하고 실행하여 목표를 이루는 사람입니다.
저는 웹 퍼블리셔로 더 발전하고 성장하여 역량을 펼쳐서 보고싶어서,
빠르게 성장하고있는 웹 퍼블리셔 분야에서 핵심인재가 되고자 이렇게 지원합니다.
스마트모바일 웹 UI UX 디자인 구현 과정을 통해 관련분야에 대해 습득하고, 직무를 통해 경험해 보았습니다.
경험과 배움을 통해 더 나아서 웹퍼블리셔에 필요한 전문가로 성장하고 싶습니다.
항상 트렌드에 민감하게 반응하고 앞서나가는 웹 퍼블리셔, 스스로 답을 찾아낼 줄 아는 웹 퍼블리셔가 되겠습니다.
